## 노드버드 풀스택 프로젝트

<br>

### 1. 프로젝트 요약

<br>

구현내용 : CRUD 게시판 형식의 페이지 구현.  
기술스택 : Next.js ,antd,Redux,Express.js,MySQL

<br>

### 2. 프로젝트 기술스택

프레임워크 : Next.js  
라이브러리 : axios, antd, styled-components, Redux,Redux-saga,React-slick

### 3.실행 방법

서버 실행 방법

```
npm start
```

클라이언트 실행 방법

```
npm i && npm run dev
```
